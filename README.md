# umi project

## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```

## plugins:
```
yarn upgrade react@^17.0.0
yarn upgrade react-dom@^17.0.0
yarn add @ant-design/pro-layout
```
